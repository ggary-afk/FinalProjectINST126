# FinalProjectINST126

# Game Name: Naasii

# Game Designer: Kenna Alexander

# game Website: https://coyoteandcrow.net/

Features:
1)This game has a simulated 12 sided dice roll for multiple dice.
2)There is a locking system that allows players to build their hand with dice that they want.
3)There is a 12 sided crow dice that will eliminate the same number dice, or will completely end your turn if it lands on 12.
4)You will score with sets of 3 or more of the same dice or with runs of 3 or more consecutive numbers.
5)Rolling a white 12 will act as a wild that can be used to complete runs or sets.
6)A graph will be generated and will visualize your run or set.

Requirements: 
1)You need to have matplotlib and Python installed.
2)launch the program by typing python main.py into your terminal.

How to play:
1:The game begins by rolling three white dice.
2:You will get asked if you want to roll to continue receiving dice. 
3:You can also score if you already have a run or set that you are happy with, please make sure you only type roll or score into your console.
4:You will have to lock at least one dice from your hand each round to save a desired die for further rounds.
5:You must type in the position of the dice that you want to lock followed by a space and another position for multiple dice like (1 3) for the first and third positioned dice.
6:If the crow die rolls a 12 then you bust and score nothing
7:If the crow die rolls and matches one of your dice then that white die will be removed.
8:The game ends after 3 rolls or will end when you choose to score, it will then check your hand for runs and sets
